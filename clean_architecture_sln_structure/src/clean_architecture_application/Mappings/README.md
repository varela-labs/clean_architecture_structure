# Mappings

## Purpose
Mapping configuration between Domain models and Application contracts (DTOs).

## Typical contents
- AutoMapper profiles (if using AutoMapper)
- Manual mapping helpers when needed

## Out of scope
- EF Core entity configurations
- UI/view mappings

## Notes
Keep mapping logic out of controllers. Prefer mapping at boundaries.
