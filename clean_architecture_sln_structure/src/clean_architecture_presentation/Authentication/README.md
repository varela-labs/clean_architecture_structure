# Authentication

## Purpose
Authentication at the transport edge (JWT bearer, identity provider integration).

## Typical contents
- JWT bearer setup
- Token validation parameters
- Auth-related options binding

## Out of scope
- Authorization rules expressed as business policy (should be in Application where possible)

## Notes
Authentication is a delivery mechanism. Keep it outside the core.
