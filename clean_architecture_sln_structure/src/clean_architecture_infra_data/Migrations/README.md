# Migrations

## Purpose
EF Core migrations that evolve the database schema over time.

## Typical contents
- Migration files and snapshots generated by EF Core tooling

## Out of scope
- Manual business logic
- Application/use case code

## Notes
Treat migrations as infrastructure artifacts. They should never introduce business policy.
